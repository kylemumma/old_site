Introduction

Python is a cross-platform, object-oriented programming language. It contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core Python can be extended for a variety of purposes by supplementing it with additional packages; for example:

  • The Requests library allows you to simply and effectively send HTTP requests with Python
  • The BeautifulSoup library allows you to easily parse through HTML code making webscraping much simpler
  • The Pillow library is a user friendly Python imaging library that allows you to easily work with images in Python
  • The NumPy adds some very advanced math functionalities to Python
  • The matplotlib library is an advanced numerical plotting library for Python, very useful for data science and data analysis

What you should already know

This guide assumes you have no previous knowledge besides a fluency with computers and the internet.

Hello World

To get started with writing Python, open IDLE and write your first "Hello world" Python code:

def hello_world(): print("Hello World!") hello_world()

Run the code and watch it unfold in the terminal!

Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A Python identifier must start with a letter or underscore (_) Subsequent characters can be letters, underscores, or numbers (0-9). Because Python is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase). There are some reserved words which you cannot use as a variable name because Python uses them for other things.

Some examples of legal names are _hits, temp99, name_of_something, and _name_name_name.

Declaring Variables

The assignment statement creates new variables and gives them values. Basic assignment statement in Python is:

variable = value

Where the equal sign (=) is used to assign value (right side) to a variable name (left side). See the following statements:

item_name = "computer" item_qty = 10 item_value = 1000.23 is_item_broken = false

Variable Scope

When you declare a variable outside of any function, it is called a global variable, because it is available to any other code in the current document. When you declare a variable within a function, it is called a local variable, because it is available only within that function.

The following would work correctly when run:

def add_numbers(): x = 10 y = 10 print(x + y)

Because all the variables are within scope. The next example however would not run correctly because the varibles x and y are out of scope.

def numbers(): x = 10 y = 10 print(x + y)

Since x and y were declared within the function, that function is the only thing that has access to them. When we try to reference the variables outside of the function, it doesn't work.
